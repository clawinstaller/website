# System Monitor

After installation, ClawInstaller stays on your system as a control panel to monitor OpenClaw's status at all times.

## Control Panel Features

### System Status

- **Gateway Status**: Check if the OpenClaw Gateway is running properly
- **Connection Info**: WebSocket address (default `ws://localhost:18789`)
- **Uptime**: How long the Gateway has been running
- **Controls**: Stop / Restart / Manual check

### Channel Settings

Add or remove communication channels anytime without reinstalling.

### AI Assistant

Got a problem? Ask the AI assistant right from the control panel — it understands your setup environment and gives targeted advice.

## Menu Bar Icon

ClawInstaller adds an icon to the macOS Menu Bar for quick access to:
- View Gateway status
- Open the control panel
- Restart services

## Auto-Refresh

The control panel auto-refreshes status every 5 seconds. You can turn off auto-refresh and check manually instead.
