# Channel Setup

Let your Agents interact with you through the messaging platforms you already use.

## Supported Channels

| Channel | Status | Description |
|---------|--------|-------------|
| Slack | Available | Best for team collaboration |
| Telegram | Available | Instant notifications, lightweight |
| Discord | Available | Community and developer use |
| LINE | Coming Soon | Popular in Taiwan & Japan |

## How to Set Up

### During Installation

Step 3 of the setup wizard lists all available channels — just toggle them on.

### After Installation

Go to Control Panel → Channel Settings to add or remove channels at any time.

## Slack Setup

1. Enable Slack in ClawInstaller
2. Click the "Connect" button
3. Authorize the OpenClaw Bot to access your Workspace
4. Select which channels to receive notifications in

## Telegram Setup

1. Enable Telegram in ClawInstaller
2. Scan the QR Code or click the link
3. Send `/start` to the Bot in Telegram
4. Binding complete

## Discord Setup

1. Enable Discord in ClawInstaller
2. Invite the Bot to your Server
3. Set which channels the Bot can access

::: tip
You can enable multiple channels at once — your Agents will respond on all active channels.
:::
