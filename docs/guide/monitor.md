# 系統監控

安裝完成後，ClawInstaller 會留在你的系統作為控制台，隨時監控 OpenClaw 的運行狀態。

## 控制台功能

### 系統狀態

- **Gateway 狀態**：確認 OpenClaw Gateway 是否正常運行
- **連線資訊**：WebSocket 連線地址（預設 `ws://localhost:18789`）
- **執行時間**：Gateway 已運行時間
- **操作按鈕**：停止 / 重新啟動 / 手動檢查

### 頻道設定

隨時新增或移除通訊頻道，不需要重新安裝。

### AI 助手

遇到問題？直接在控制台問 AI 助手，它了解你的安裝環境，能給出針對性的建議。

## 系統列圖示

ClawInstaller 安裝後會在 macOS 系統列（Menu Bar）顯示圖示，點擊即可快速：
- 查看 Gateway 狀態
- 開啟控制台
- 重啟服務

## 自動刷新

控制台預設每 5 秒自動刷新狀態，你也可以關閉自動刷新改為手動檢查。
