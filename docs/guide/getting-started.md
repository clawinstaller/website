# 快速開始

歡迎使用 ClawInstaller！只需 3 分鐘，你就能擁有一個由 3 位 AI Agent 組成的工作團隊。

## 系統需求

| 項目 | 最低需求 |
|------|----------|
| macOS | 14.0 (Sonoma) 以上 |
| Node.js | 18.0 以上 |
| 套件管理器 | npm / pnpm / yarn / bun（任一） |
| 磁碟空間 | 500 MB 可用空間 |

## 下載安裝

### 方式一：直接下載（推薦）

前往 [GitHub Releases](https://github.com/clawinstaller/claw-installer/releases) 下載最新版 `.dmg` 檔案。

### 方式二：Homebrew

```bash
brew tap clawinstaller/tap
brew install --cask clawinstaller
```

## 安裝流程

ClawInstaller 會引導你完成以下步驟：

### 1. 環境檢查

安裝精靈會自動偵測：
- ✅ macOS 版本
- ✅ Node.js 版本
- ✅ 套件管理器（npm/pnpm/yarn/bun）
- ✅ 磁碟空間
- ✅ 網路連線

如果有缺少的項目，會提供一鍵修復按鈕。

### 2. 安裝 OpenClaw

一鍵安裝 OpenClaw CLI 到你的系統。安裝過程會在終端機即時顯示進度。

### 3. 頻道設定

選擇你想啟用的通訊頻道：
- **Slack** — 團隊協作
- **Telegram** — 即時通知
- **Discord** — 社群互動
- **LINE** — 台灣最常用的通訊軟體

### 4. 完成！

安裝完成後，你的三位 Agent 就上線了。可以：
- 開啟終端機輸入 `openclaw` 開始對話
- 使用控制台監控系統狀態
