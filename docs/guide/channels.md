# 頻道設定

讓你的 Agent 透過你習慣的通訊軟體跟你互動。

## 支援的頻道

| 頻道 | 狀態 | 說明 |
|------|------|------|
| Slack | ✅ 可用 | 適合團隊協作 |
| Telegram | ✅ 可用 | 即時通知、輕量 |
| Discord | ✅ 可用 | 社群與開發者 |
| LINE | 🔜 即將推出 | 台灣用戶最愛 |

## 設定方式

### 在安裝過程中

安裝精靈的第三步會列出所有可用頻道，開啟 toggle 即可啟用。

### 安裝後設定

進入控制台 → 頻道設定，隨時新增或移除頻道。

## Slack 設定

1. 在 ClawInstaller 中啟用 Slack
2. 點擊「連接」按鈕
3. 授權 OpenClaw Bot 存取你的 Workspace
4. 選擇要接收通知的頻道

## Telegram 設定

1. 在 ClawInstaller 中啟用 Telegram
2. 掃描 QR Code 或點擊連結
3. 在 Telegram 中對 Bot 發送 `/start`
4. 完成綁定

## Discord 設定

1. 在 ClawInstaller 中啟用 Discord
2. 邀請 Bot 到你的 Server
3. 設定 Bot 可以存取的頻道

::: tip
你可以同時啟用多個頻道，Agent 會在所有啟用的頻道上回應。
:::
